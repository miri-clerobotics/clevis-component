# 노드 기본 동작 가이드

우리 프로그램에서 사용하는 핵심 노드 컴포넌트의 동작 가이드입니다.  
아래 화면을 통해 실제 프로그램과 동일한 인터랙션을 미리 체험해볼 수 있습니다.

### 인터랙션 테스트 베드
* 노드를 **클릭**하면 선택 상태가 전환됩니다.
* 노드를 **우클릭**하면 컨텍스트 메뉴가 나타납니다.
* 포트 라벨을 **드래그**하여 순서를 변경할 수 있습니다.
